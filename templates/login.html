Welcome to Audio File Manager!

Guide:

  • /list
  • /info
  • /download
  • /post
  • /delete

/list

List your files. Optional filtering by duration length.

Optional keywords: maxduration, minduration

e.g. http://localhost:5000/list?minduration=25&maxduration=30

(command line) $ curl "http://localhost:5000/list?minduration=25&maxduration=30"

/info

Get file metadata. If name is not passed, returns metadata for all files.

Optional keyword: name

e.g. http://localhost:5000/info?name=myfile.wav

(command line) $ curl "http://localhost:5000/info?name=myfile.wav"

/download

Download a file from your user directory.

Required keyword: name

e.g. http://localhost:5000/download?name=myfile2.wav

(command line) $ curl "http://localhost:5000/download?name=myfile2.wav"

/post

Add a new file to your user directory. Valid file types include: .wav, .mp3, .aiff

e.g. http://localhost:5000/post then select file

(command line) $ curl -F "file=@myfile.wav" http://localhost:5000/post

/delete

Remove a file from your directory

Required keyword: name

e.g. http://localhost:5000/delete?name=myfile.wav

(command line) $ curl "http://localhost:5000/delete?name=myfile.wav"
